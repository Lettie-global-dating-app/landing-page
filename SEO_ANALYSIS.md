- 자연스러운 키워드 배치
- 동의어 및 관련어 포함

### 3. 내부 링크 구조 (Internal Linking)
✅ **점수: 85/100**
- 블로그 목록 ↔ 개별 포스트

- 메인 페이지 ← 블로그

- 예상 노출 증가: +150%

- 예상 백링크: 10-20개/월
